ACBrCHQClass

Declaração

type TACBrCHQClass = class(TObject)

Descrição

Classe generica de Cheques, nao implementa nenhum modelo especifico, apenas declara a Classe. NAO DEVE SER INSTANCIADA. Usada apenas como base para as demais Classes de Imp.Cheques como por exemplo a classe TACBrCHQChronos
