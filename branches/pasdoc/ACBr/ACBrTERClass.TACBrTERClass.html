ACBrTERClass

Declaração

type TACBrTERClass = class(TObject)

Descrição

Classe generica de TERMINAL, nao implementa nenhum modelo especifico, apenas declara a Classe. NAO DEVE SER INSTANCIADA. Usada apenas como base para as demais Classes de TERMINAL como por exemplo a classe TACBrTERWilbor
