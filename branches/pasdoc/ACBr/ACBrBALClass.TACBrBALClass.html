ACBrBALClass

Declaração

type TACBrBALClass = class(TObject)

Descrição

Classe generica de BALANCA, nao implementa nenhum modelo especifico, apenas declara a Classe. NAO DEVE SER INSTANCIADA. Usada apenas como base para as demais Classes de BALANCA como por exemplo a classe TACBrBALFilizola
