TACBrECFExibeErroEvent = procedure(Erro : Exception) of object ;

Evento para o usuário exibir os erros encontrados pela classe TACBrECFClass. Se o evento OnMsgErro NAO for programado a Classe TACBrECFClass exibirá as Msg de erro através de Exceçoes. Se o evento OnMsgErro for programado a Classe nao exibe nenhuma msg de erro, que deverao ser tratados dentro deste evento
