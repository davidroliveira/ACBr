Procedure SetModelo( Const Value : TACBrTERModelo );

Cada Linha é Definida na forma [L][TT][...] Onde [L] é o número da Linha do Terminal ['1', '2']; [TT] é o número do Terminal ['00'..'99']; [...] é a String que ainda falta para girar.
