Unit

ACBrGAVClass

Declaração

type TACBrGAVClass = class(TObject)

Descrição

Classe generica de Gaveta, nao implementa nenhum modelo especifico, apenas declara a Classe. NAO DEVE SER INSTANCIADA. Usada apenas como base para as demais Classes de Gavetas como por exemplo a classe TACBrGAVMenno
